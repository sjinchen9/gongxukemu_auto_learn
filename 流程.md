# 自动学习脚本 v4.0 完整工作流程

## 一、启动阶段

1. **版本检查** → 非 Python 3.11 则报错退出
2. **minimize_console()** → CMD窗口缩小为 700×180，居中贴屏幕顶部
3. **初始化**：
   - `OCREngine` 首次加载 PaddleOCR（下载模型，约30秒）
   - `Detector` 初始化（OCR缓存）
   - `AutoLearner` 状态机 → `state=None`（首次自检）
4. 等待3秒，立即执行首次 `_scan()`

---

## 二、主循环 run()

```python
while clicks < MAX_CLICKS(50):
```

### 手动模式（s键触发）

```
self._scan() → 每0.1秒检查按键 → 10次后结束
按 a 切回自动，按 q 退出
```

### 自动模式

```
等待 interval 秒（每秒检查按键）
前3轮: 15秒   视频模式: 60秒   选课模式: 10秒
等待结束后执行 self._scan()
```

## 三、单轮扫描 _scan()

### Step 1: 截屏 + OCR

- `pyautogui.screenshot()` 截取全屏
- PaddleOCR 识别文字 → `[{text, cx, cy, conf}, ...]`
- 耗时约 8-10 秒（det 检测框 + rec 文字识别）
- 过滤 conf<0.5 的低质量结果
- `Detector.scan()` 缓存结果，后续检测都从缓存读取

### Step 2: 页面可见性检查 has_page()

- **优先 OCR**：缓存中有培训关键词（下一节/关闭/确定/未完成等）或 OCR 词数 > 20
- **兜底颜色**：蓝色标题栏 >1000 像素 或 画面暗色 >20%
- → 不通过 → 跳过本轮（防止在非培训页面乱点）

### Step 3: 首次状态检测（state=None 时）

```
有 "关闭" 或 "下一节"  → State.VIDEO
有 "未完成"            → State.COURSE
都没有                 → dark_ratio>0.3 判 VIDEO，否则 COURSE
```

### Step 4: 全局弹窗优先检测 ★最高优先级★

- OCR 搜 "确定/确认" + 颜色验证蓝色按钮 + 白色对话框 + 遮罩
- 去重：同一位置（±10px）不重复点击
- 点击 → 等待5秒 → `needs_refresh=True` → 本轮结束
- 不通过 → 继续后续流程

### Step 5: 每轮自检修正在状态

三个信号：close存在 / next存在 / dark_ratio>0.3

| 信号 | 当前状态 | 动作 |
|------|----------|------|
| 信号→视频 | state=COURSE | 修正为 VIDEO（清除刷新标志） |
| 信号→选课 | state=VIDEO | 修正为 COURSE（标记需刷新） |

### Step 6: 选课页刷新

当 `needs_refresh=True` 且 `state=COURSE` 时：
1. 右键屏幕中央获取焦点
2. F5 刷新 → 等10秒
3. PageDown 滚动

### Step 7: 状态分发

```
state==VIDEO  → _do_video()
state==COURSE → _do_course()
```

---

## 四、视频页处理 _do_video()

### 情况A：检测到"下一节"

- OCR 搜 "下一节"（排除顶部12%和底部12%）
- 取第一个匹配 → `_do_click(next)`
- → 验证: 1.5秒后画面变暗 > 20%

### 情况B：没"下一节" + 画面大面积全黑 > 50%

- → 视频播完了
- 搜右上角 "关闭" 按钮 → `_do_click(close)`
- → 验证: 1.5秒后出现 "确定/确认"
- → 链式处理: 等2秒 → 找"确定" → 点击弹窗 → 等5秒 → 标记刷新
- 关闭找不到 → 等待下一轮

### 情况C：没下一节 + 画面没全黑

- → 视频播放中，miss_count=0，等下一轮

---

## 五、选课页处理 _do_course()

### Step 1: 搜"未完成"

- OCR 搜 "未完成" + 排除旁边有 "已完成" 的（200px水平 + 30px垂直）
- 排除 `done_positions` 中已点击过的位置
- 按Y坐标排序（靠上的优先）
- 找到 → `_do_click(incomplete, 坐标=未完成文字位置)`
  - → 验证: 1.5秒后出现 "关闭" 按钮 → 成功进入课程
  - 成功: `done_positions.add(位置)`, `state→VIDEO`
  - 失败: `done_positions.add(位置)`, 标记避免重试（误点击保护）

### Step 2: End键跳转到页底

- `course_scrolls==0` 时触发
- 右键获取焦点 → `press('end')` → `course_scrolls=1`

### Step 3: 底部再搜"未完成"

- 重新截屏+OCR → `find_incomplete()`
- 找到 → 同上流程

### Step 4: 搜翻页栏

- 在右下角（60%-98%宽, 82%-98%高）搜索数字和 `>` `>>` `›` `»` 符号
- 用颜色检测找到蓝色高亮的当前页码数字
- 如果当前页是 N → 找数字 N+1 并点击
- 如果 N+1 未显示（如最后一页）→ 点击 `>` 符号
- `>` 符号在任何页面都可以用来翻到下一页

### Step 5: Home回顶+逐页找选项卡

- 右键获取焦点 → `press('home')`
- 循环6次:
  - 截屏OCR → `find_tabs()`（"必修课/选修课"）
  - 找到 ≥2 个选项卡 → 停止
  - 没找到 → PageDown 翻页
- 找到选项卡后: `tab_index` 循环切换 → 点击 → `course_scrolls=0`
- 六次都没找到 → "暂无目标，继续等待"

## 六、验证系统 _verify_click()

每次 `_do_click` 后，等 **1.5秒** 验证:

| 点击类型 | 验证方法 | 通过标准 |
|----------|----------|----------|
| `next` (下一节) | 画面暗色比例 | > 20% |
| `incomplete` (未完成) | 出现"关闭"按钮 | 找到了 |
| `close` (关闭) | 出现"确定/确认" | 找到了 |
| `confirm` (确定) | "确定/确认"消失 | 找不到了 |
| `tab` (选项卡) | 直接通过 | - |

### 验证失败处理

- `state→None`（下轮重新自检）
- `course_scrolls→0`
- 保存 `DEBUG_before` + `DEBUG_after` 截图到 screenshots 目录

## 七、OCR 检测器方法一览

所有方法都从 OCR 缓存中查询，不需要额外截屏。

| 方法 | 搜索词 | 搜索区域 |
|------|--------|----------|
| `find_next()` | "下一节" | 全屏（排除顶部12%） |
| `find_close()` | "关闭" | 右上角 85-100%宽, 0-20%高 |
| `find_confirm()` | "确定"/"确认" + 蓝色验证 | 中部 20-80%宽, 25-75%高 |
| `find_incomplete()` | "未完成" + 排除"已完成"（200px内） | 10-90%宽, 35-85%高 |
| `find_tabs()` | "必修课"/"选修课"/"必修"/"选修" | 5-95%宽, 10-45%高 |
| `find_pagination()` | 数字/`>`/`>>`/`›`/`»` + 蓝色当前页 | 右下角 60-98%宽, 82-98%高 |
| `has_page()` | OCR关键词 + 蓝色标题栏 + 暗色比例 | - |

## 八、弹窗上下文验证 _verify_confirm_context()

保留 v3 结构检测，确保 OCR 识别的"确定"是真的弹窗按钮：

1. **白色对话框检测**：屏幕中部 25%-65%高, 20%-80%宽，白色像素 > 5000
2. **遮罩检测**：白色对话框上方30px区域，暗像素 > 30%
3. **按钮颜色验证**：匹配位置的局部区域，蓝色像素 > 8%

三者全部通过 → 确认是弹窗

## 九、完整学习循环示例

```
[启动] state=None
   ↓
第1轮 _scan: 自检→VIDEO 模式
   ↓
视频播放中... 等60秒...
   ↓
第2轮: 搜到"下一节" → 点击 → 验证成功
   ↓
下一节视频播放中... 等60秒...
   ↓
...循环直到全部小节学完...
   ↓
第N轮: 画面全黑→"关闭"→点击→链式"确定"→state→COURSE
   ↓
第N+1轮: needs_refresh→F5→搜"未完成"→点击→state→VIDEO
   ↓
新课程视频播放中... 回到循环起点
```

### 如果选课页找不到未完成

```
搜"未完成"未果
  → End键跳到底部
  → 底部再搜未完成
  → 搜翻页栏（下一页）
  → Home键回到顶部
  → 逐页找选项卡（必修课/选修课）
  → 切换选项卡后重新搜未完成
```
